# plotWav.py

This script is designed to conveniently display `.wav` files with more than one channels of input.

## How it works

1. Modify the value of `data = wavfile.read()` on **line 23**.
   - Set this to the path + data file name of the `.wav` file you want to view.

2. Run the Python script to visualize the `.wav` file.

# plotWavApp_v1.py

This script is designed to conveniently display `.wav` files with one or more channels of input.
It supports a simple user friendly interface.

## How it works

1. Click the `Upload`Button.
   - choose the  `.wav` file you want to view.

2. Click `View Data` Button.
